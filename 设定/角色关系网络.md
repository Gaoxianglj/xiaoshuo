## 一、主角层

### 莉莉安（Lilian）—— 女主角

```
身份：孤儿 → 精灵养女 → 流浪儿 → 阿尔维斯弟子 → 艾瑞安学院第一位平民学生
      → 卢卡斯的未婚妻（被迫接受）→ 斯托克顿领的寄居者
外貌：纯白长发 / 淡紫色瞳孔 / 苍白肤色 / 身形瘦弱
年龄：约16岁（故事开始时）
魔力水平：极高（实战魔法同级第一）
性格：表面温和顺从（后天练就的保护面具）/ 内在敏感坚韧 / 习惯性隐藏 / 对善意既渴望又恐惧
核心矛盾：不完全属于人类世界，也不能回到精灵世界

**【深层内心与学院篇情感弧线】**

莉莉安因为童年的阴影和白发，内心一直是自卑且缺乏安全感的。
她很想拥有一个归宿——一个能够绝对包容她的归宿。
这源于她从来没有真正拥有过"家"的体验。

被师傅送往学院时——不安但顺从。她以为能和师傅过一辈子平凡的隐居生活，
但在师傅要送走她时没有提出反对。她学会伪装内心——不想让师傅担心，
不想让他觉得自己是个烦人的小姑娘。

学院初期——微弱的希望。玛琳的出现让她第一次体验到"有朋友"的感觉。
她在学院中的魔法进步和教授们的认可给了她一种陌生的温暖。
她开始——短暂地——相信自己或许可以在这里找到一个位置。

**被陷害时——震惊而非醒悟**。
在地下室面对玛琳的指控时，莉莉安并没有"立刻想通一切"。
她的大脑拒绝处理这些信息——她仍然在内心试图为玛琳寻找解释。
直到被关入监狱后，她才开始怀疑。但她逼着自己相信"玛琳一定是被胁迫的"。
这是她唯一的心理支撑——如果连这个解释都没有了，她就什么都没有了。

**在狱中——渐进的崩溃**。
塞西莉亚以半真半假的话语逐步瓦解她的防线——
其中有些细节明显是假的（给了她否定整段话的理由），
但核心事实无法否认（玛琳接近她确实有目的）。
真正的摧毁来自于不得不面对"起点是假的"这个事实——
不管玛琳后来是否真的把她当成朋友。

**逃离后——最后的希望熄灭**。
回到师傅的木屋发现一切已被焚毁——她以为师傅受了她的牵连。
这是对她"被诅咒"的自我认知的最终印证。

**全面崩溃**。
在多重打击（友人的利用/背叛、肉体的虐待、心理的瓦解、师傅木屋被焚）下，
莉莉安的心理防线彻底坍塌。她被扭曲了认知——认为自己确实应该被这样对待，
因为自己给所有亲近之人都带来了不幸。

**离开学院时**。
重新变回了那个雨天缩在巷子里的流浪的小女孩。
认为自己是被诅咒的人，触碰了不该触碰之物的人。
故事开始时（第一章-第二章），莉莉安的心已经崩碎——
她是被塞入马车送到斯托克顿领的。跪在卢卡斯面前说出那些卑微的话时，
不是自虐，而是一个已经放弃一切的人最后的、麻木的诚实。

小说的第一大部分主题：爱与亲情与归宿——是治愈少女已然崩碎的内心。
```

**关系**：
- ← 精灵们："爱子"，被精灵从婴儿时期抚养长大的孩子
- ← 母亲（已故）：在她出生后将她带入原初之地，在湖畔因伤势过重去世，葬于湖畔花丛
- ← 阿尔维斯·洛恩：师傅，在雨天城市中发现并收留了她（约7年前），教授魔法约七年。莉莉安敬爱他，也因不想让他担心而学会了隐藏内心
- ← 卢卡斯·斯托克顿：未婚夫（被迫接受）。第二章首次见面时莉莉安以最卑微的姿态跪在他面前，内心相信自己的存在只会给他人带来麻烦
- ← 玛琳·费尔德：学院中唯一主动接近她的同学（动机存疑）。莉莉安曾视她为朋友，后来遭到背叛——这是压垮她内心的最后一根稻草
- ← 茉莉·凯恩：魔法学院同班，因新生代表位置被抢而敌视（但"其实不讨厌她"），在父亲授意下虐待莉莉安
- ← 艾尔文：精灵学院院长，曾为她解围并对她的试卷表现出特殊兴趣
- ← 菲莱尔：一只给自己起了名字的精灵，对她有特殊的关注和喜爱
- ← 萨拉·约翰逊：斯托克顿家女仆长，第一个用温暖拥抱她的人类。莉莉安因长期被排斥而僵硬，但那拥抱是她需要的
- ← 艾尔伯·斯托克顿：未婚夫之父，斯托克顿男爵
- → 被大多数贵族同学排斥/畏惧/敌视

### 卢卡斯·斯托克顿（Lucas Stockton）—— 男主角

```
身份：斯托克顿男爵继承人 / 政商学院学生 / 异世界转生者（灵魂来自"地球"）
外貌：（原始资料未详细描述）
年龄：约16-17岁
魔力水平：掌握治愈魔法等
性格：理智沉稳 / 善于观察 / 有正义感 / 不擅长与女性交往 / 有商业头脑
核心矛盾：拥有神明植入的"剧本"知识但现实已偏离 / 理性计划与逐渐产生的情感
```

**转生的真相**：
- 卢卡斯前世是地球上的一名普通高中生，死于意外
- 灵魂回归亚空间后被这个世界的神明选中
- 神明将一条未来时间线伪装为"乙女游戏剧情"的记忆植入他的灵魂
- 目的：让他拥有关于这个世界的"前置知识"，以干预命运的走向，阻止世界毁灭
- 所谓的"乙女游戏"从未存在过——那是神明伪造的记忆
- 他的"剧本知识"本质上是真实世界的命运线预览，会随着他的干预逐渐偏离

**关系**：
- ← 艾尔伯·斯托克顿：父亲，斯托克顿男爵。对父亲的处境心怀同情与责任感。
- ← 安娜·史密斯（已故）：母亲，葬于山顶
- ← 亚特伍德·斯托克顿：爷爷，第一代斯托克顿男爵，骑士出身，参与圣战立功受封
- ← 莉莉安：未婚妻（由福克斯子爵强行指婚）。卢卡斯在认出她是"女主角"后震惊不已。他在第二章试图以温和的方式对待她，却发现她的内心已然崩碎。
- ← 萨拉·约翰逊（安娜）：家中的女仆长，信任的助手（注：第一章中名为萨拉，第二章中以"安娜"称呼——可能是同一人，也可能是有两个女仆，需后续统一）
- → 福克斯子爵：敌视，福克斯子爵试图通过婚约渗透斯托克顿领
- → 学院中的其他"游戏男主角"：卢卡斯在"剧本"中知道他们，但现实已偏离

## 二、核心关系层

### 阿尔维斯·洛恩（Alvis Lorne）—— 师傅

```
身份：前主教之首 / 隐居的魔法研究者 / 实力至上主义者
外貌：苍老，黑衣银纹，袖口与衣领带有教国上层银纹
性格：外冷内热 / 狂热的研究者 / 不擅表达感情 / 对教廷迂腐持批判态度
      盲点：认为莉莉安会因魔法实力而自信，未察觉她内心深处的自卑与脆弱
魔法水平：主教级（可能掌握一种"神迹"魔法）
当前状态：已返回教廷参加秘密魔法研究项目（莉莉安入学后），安然无恙
```

**离开教廷的真相**：
- 阿尔维斯是**主动辞去主教之首一职**，并非叛逃或被迫害
- 原因：代表进步派与保守派产生理念分歧 → 保守派占上风 → 愤然辞职
- 进步派理念：社会制度应取决于生产力，魔导具与科学工程学的进步要求打破贵族对知识和教育的垄断，让有实力的人不再被血脉束缚
- 七位主教之间遵守传统以同门师兄师弟（师姐师妹）相称，彼此无血海深仇
- 其余主教尊重了阿尔维斯的选择，未曾追捕他

**莉莉安入学的交易**：
- 阿尔维斯写信给现任主教们请求让莉莉安入学
- 交换条件：阿尔维斯返回教廷参加秘密魔法研究项目
- 他的动机：1) 莉莉安有无尽的可能性，应获得更好的教育 2) 让贵族子弟看清楚血脉不是一切 3) 他的实力至上主义
- 返回教廷后对莉莉安在学院的遭遇全然不知，盲目自信徒弟的实力

**住处被毁的真相**：
- 其余主教做的痕迹消除工作——不想让阿尔维斯在外留下痕迹，并非针对莉莉安

**关系**：
- ← 莉莉安：弟子，在七年中教会了她魔法和生活，是改变她命运的人
- → 教廷：前主教之首 → 主动辞职 → 隐居 → 以交易方式返回（参与秘密研究）
- → 其他六位主教：曾经的同事，以同门师兄师弟相称，虽理念不合但无仇恨

**叙事功能**：
- 莉莉安的魔法导师和生活拯救者
- 教廷内部的知情人
- 莉莉安寻找师傅的悬念线（她不知道师傅在教廷）
- 可能与艾瑞安·赛拉斯被弑的真相有关

### 艾尔文（Elwin）—— 精灵学院院长

```
身份：精灵学院院长 / 九大学院长之一
外貌：棕发棕瞳 / 皮肤粗糙 / 眼角细纹 / 着装随意（领扣不系）
表面形象：慵懒、随意、下午常醉醺醺
内在：极其敏锐、洞察力强、善于用不经意的闲聊包裹精准的试探
试探手法：从不直接追问，而是用慵懒的语气在闲聊中插入关于精灵的精准问题——
          在莉莉安差点脱口而出时也不追问，只是微笑，让莉莉安自己意识到危险
```

**关系**：
- ← 莉莉安：在新生典礼后第一次接触她，此后以"偶遇"的形式多次试探。
  审阅莉莉安的精灵语试卷后发现不寻常之处（她的精灵语水平远超学院最顶尖研究者）。
  他的试探方式让莉莉安对他产生了防备——"这个人在探究她最深的秘密"。
- ← 萨兰：精灵学院的老师，实际打理学院事务的人
- → 莉莉安："我已经开始后悔放你去别的学院了"
- → 学院体制：对学院作为"权力角斗场"持批判态度
- → 玛琳的请假：莉莉安代玛琳请假时，艾尔文察觉到了不对劲但未深究

**叙事功能**：
- 在学院中庇护莉莉安的潜在力量
- 发现莉莉安特殊性的关键人物——目前唯一对莉莉安与精灵真实关系有系统性怀疑的人类
- 精灵研究的专家——可能在后续揭示精灵与人类关系方面发挥重要作用
- 他的试探构成了莉莉安在学院日常中的一条持续紧张的暗线

### 菲莱尔（Fier）—— 精灵

```
身份：一只给自己起了名字的精灵（绝大多数精灵没有名字）
特征：比其他精灵更有个性、更活泼、更愿意与人类互动
      对莉莉安有近乎偏执的保护欲
态度：对人类持嘲讽态度——"愚笨的行为符合精灵对于人类的定义"
与莉莉安的关系：特别关注，称莉莉安为"我们的爱子"
```

**精灵的诚信感知**：
精灵重视契约与诚信。它们能本能地"闻出"人类身上的气味——
诚实守信的人气味清新，说谎背信的人散发着臭味。
但这种感知是笼统的——精灵无法分辨谎言的具体内容，
只能感知到一个人整体上是否"臭"。
因为大多数人类都擅长说谎，所以在精灵闻来，大多数人类都是"臭的"——
这也是精灵对人类态度冷淡的原因之一。

**关系**：
- ← 莉莉安："爱子"
- → 玛琳·费尔德：从第一次见面就表现出厌恶——"那个蓝头发的人类，气味不太好"
  （因为玛琳确实在对莉莉安撒谎，但菲莱尔无法具体指出在哪里）
- → 原初之地年长精灵：在发现玛琳的阴谋后试图杀死玛琳，
  被年长精灵强制"软禁"，心中对它们的恨意达到顶点
  （伏笔：精灵内部分裂）
- → 人类整体：轻蔑（大多数人类闻起来都"臭"）
- → 其他精灵：有名字是独特之处

**叙事功能**：
- 精灵视角的代言者（表达精灵对人类的态度）
- 莉莉安与精灵世界的连接线
- 通过"气味"设定提供关于玛琳的暗示（读者和莉莉安一样不会立即解码）
- 与年长精灵的冲突——精灵内部"干涉vs不干涉"原则之争的体现
- 喜剧调剂（性格活泼）与悲剧载体（被束缚时的无力）

### 原初之地年长精灵（精灵长老的使者）

```
身份：来自原初之地的年长精灵，精灵长老会的意志执行者
特征：比普通精灵更古老、更强大，通常不显现实体形态
使命：监视莉莉安身边精灵与人类的互动，防止精灵再次打破"不干涉人类"的约定
立场：不是冷血，而是对规则的绝对遵守——为了防止更大的连锁反应
```

**为何存在**：
- 莉莉安儿时那次火灾事件（精灵因佃农殴打莉莉安而放火烧死施暴者）之后，
  精灵长老会决定派遣年长精灵跟随莉莉安
- 不是因为莉莉安有危险——而是因为精灵们太爱她了
- 莉莉安浑身充满原初之地的气息、高洁而善良，精灵们会本能地喜欢上她
- 这种喜欢太过强大，强大到可能会再次打破不干涉的约定
- 年长精灵的任务是：在关键时刻约束住其他精灵，不让他们为了莉莉安而干涉人类世界

**关键行为**：
- 在菲莱尔发现玛琳的阴谋并试图杀死玛琳时，年长精灵控制住了菲莱尔
- 将菲莱尔等亲近莉莉安的精灵强制"软禁"，禁止它们干涉
- 在莉莉安被囚禁时，仅允许在不直接"干涉"的前提下提供最小帮助——
  锁门的锁弹开、辉石灯暂时熄灭、巡逻狱卒恰好转向另一条走廊……
  这些微小的"巧合"是年长精灵允许精灵们帮助莉莉安越狱的极限
- 莉莉安崩溃并被送出学院后，释放了菲莱尔等被软禁的精灵

**与菲莱尔等精灵的矛盾（伏笔）**：
- 菲莱尔等精灵被释放后，对原初之地"老古董"精灵的恨意达到顶点
- 它们愤然离去——精灵内部因莉莉安事件产生了前所未有的分裂
- 这个裂痕将在后续剧情中产生深远影响

### 同班/同级

#### 玛琳·费尔德（Marin Field）
```
身份：费尔德男爵第三女 / 魔法学院同班 / 凯恩千金的卧底
外貌：蓝发蓝眼
家世：外六圈准男爵（"贫穷"的新贵族，家族面临破产危机）
性格：表面直率、开朗、主动；内在背负巨大压力、内心矛盾
      与莉莉安相处的几个月中，真心对其产生友情——
      但这让她的任务变得更加痛苦。
```
- 受凯恩家族以经济援助为条件，主动接近莉莉安进行监视
- 在最初几个月中被塑造为莉莉安"唯一的、真正的朋友"——两人的友情在表面上温暖真挚
- 积极撮合莉莉安与边境小贵族订婚：表面是为了莉莉安"提升阶级"，实则是希望莉莉安
  能主动退学，避免事情走向阴谋与暴力——"不想走到那一步"
- 莉莉安屡次拒绝求亲让她始料未及，内心的焦虑与对莉莉安的感情冲突加剧
- 在两周共同生活期间，对莉莉安的友情达到顶点——这段时光对她的内心折磨极为深刻
- 最终执行陷害计划后选择退学，前往海滨城市以平民身份度过余生（伏笔：可能回归）
- 名言："……我配后悔吗？"——这不是问句，是她对自己的判决
- 核心矛盾：她确实利用了莉莉安，但也确实真心把莉莉安当成了朋友。
  这种"既是背叛者也是朋友"的双重身份，使她成为故事中最复杂的角色之一

#### 茉莉·凯恩（艾蕾诺·凯恩）
```
身份：凯恩伯爵千金 / 魔法学院同班
外貌：紫发
家世：内三圈老伯爵家族——凯恩家族（银矿），强硬维护贵族特权
立场：被父亲要求执行打压莉莉安的计划；
      本人并非恶人，但对莉莉安抱有复杂情感——敌意、嫉妒、
```
