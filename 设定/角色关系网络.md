## 一、主角层

### 莉莉安（Lilian）—— 女主角

```
身份：孤儿 → 精灵养女 → 流浪儿 → 阿尔维斯弟子 → 艾瑞安学院第一位平民学生
      → 卢卡斯的未婚妻（被迫接受）→ 斯托克顿领的寄居者
外貌：纯白长发 / 淡紫色瞳孔 / 苍白肤色 / 身形瘦弱
年龄：约16岁（故事开始时）
魔力水平：极高（实战魔法同级第一）
性格：表面温和顺从（后天练就的保护面具）/ 内在敏感坚韧 / 习惯性隐藏 / 对善意既渴望又恐惧
核心矛盾：不完全属于人类世界，也不能回到精灵世界

【深层内心】
莉莉安因为童年的阴影和白发，内心一直是自卑且缺乏安全感的。
她很想拥有一个归宿——一个能够绝对包容她的归宿。
这源于她从来没有真正拥有过"家"的体验。

被师傅送往学院时她非常不安，她以为能和师傅过一辈子平凡的隐居生活。
但在知道这里也不是自己的归宿后，她像曾经数次离开寄宿家庭一样，选择了不做纠缠，开始了下一处"流浪"。
只不过这一次她学会了伪装自己的内心——不想让师傅担心，不想让他觉得自己是个烦人的小姑娘。

学院生活曾一度让她觉得还算开心，但凯恩伯爵千金的虐待让自卑和不安再次浮现。
当跑回与师傅隐居的地方发现被毁后陷入绝望，当再知道友人的背叛时便彻彻底底地绝望了。
认为自己永远无法奢求什么归宿，重新变回了那个雨天缩在巷子里的流浪的小女孩。
认为自己是被诅咒的人，触碰了不该触碰之物的人。

故事开始时（第一章-第二章），莉莉安的心已经崩碎。
她跪在卢卡斯面前说出那些卑微的话时，不是自虐，而是真心认为自己只配得到这样的待遇。
小说的第一大部分主题：爱与亲情与归宿——是治愈少女已然崩碎的内心。
```

**关系**：
- ← 精灵们："爱子"，被精灵从婴儿时期抚养长大的孩子
- ← 母亲（已故）：在她出生后将她带入原初之地，在湖畔因伤势过重去世，葬于湖畔花丛
- ← 阿尔维斯·洛恩：师傅，在雨天城市中发现并收留了她（约7年前），教授魔法约七年。莉莉安敬爱他，也因不想让他担心而学会了隐藏内心
- ← 卢卡斯·斯托克顿：未婚夫（被迫接受）。第二章首次见面时莉莉安以最卑微的姿态跪在他面前，内心相信自己的存在只会给他人带来麻烦
- ← 玛琳·费尔德：学院中唯一主动接近她的同学（动机存疑）。莉莉安曾视她为朋友，后来遭到背叛——这是压垮她内心的最后一根稻草
- ← 茉莉·凯恩：魔法学院同班，因新生代表位置被抢而敌视（但"其实不讨厌她"），在父亲授意下虐待莉莉安
- ← 艾尔文：精灵学院院长，曾为她解围并对她的试卷表现出特殊兴趣
- ← 菲莱尔：一只给自己起了名字的精灵，对她有特殊的关注和喜爱
- ← 萨拉·约翰逊：斯托克顿家女仆长，第一个用温暖拥抱她的人类。莉莉安因长期被排斥而僵硬，但那拥抱是她需要的
- ← 艾尔伯·斯托克顿：未婚夫之父，斯托克顿男爵
- → 被大多数贵族同学排斥/畏惧/敌视


- 神明将一条未来时间线伪装为"乙女游戏剧情"的记忆植入他的灵魂
- 目的：让他拥有关于这个世界的"前置知识"，以干预命运的走向，阻止世界毁灭
- 所谓的"乙女游戏"从未存在过——那是神明伪造的记忆
- 他的"剧本知识"本质上是真实世界的命运线预览，会随着他的干预逐渐偏离

**关系**：
- ← 艾尔伯·斯托克顿：父亲，斯托克顿男爵。对父亲的处境心怀同情与责任感。
- ← 安娜·史密斯（已故）：母亲，葬于山顶
- ← 亚特伍德·斯托克顿：爷爷，第一代斯托克顿男爵，骑士出身，参与圣战立功受封
- ← 莉莉安：未婚妻（由福克斯子爵强行指婚）。卢卡斯在认出她是"女主角"后震惊不已。他在第二章试图以温和的方式对待她，却发现她的内心已然崩碎。
- ← 萨拉·约翰逊（安娜）：家中的女仆长，信任的助手（注：第一章中名为萨拉，第二章中以"安娜"称呼——可能是同一人，也可能是有两个女仆，需后续统一）
- → 福克斯子爵：敌视，福克斯子爵试图通过婚约渗透斯托克顿领
- → 学院中的其他"游戏男主角"：卢卡斯在"剧本"中知道他们，但现实已偏离

## 二、核心关系层

### 阿尔维斯·洛恩（Alvis Lorne）—— 师傅

```
身份：前主教之首 / 隐居的魔法研究者 / 实力至上主义者
外貌：苍老，黑衣银纹，袖口与衣领带有教国上层银纹
性格：外冷内热 / 狂热的研究者 / 不擅表达感情 / 对教廷迂腐持批判态度
      盲点：认为莉莉安会因魔法实力而自信，未察觉她内心深处的自卑与脆弱
魔法水平：主教级（可能掌握一种"神迹"魔法）
当前状态：已返回教廷参加秘密魔法研究项目（莉莉安入学后），安然无恙
```

**离开教廷的真相**：
- 阿尔维斯是**主动辞去主教之首一职**，并非叛逃或被迫害
- 原因：代表进步派与保守派产生理念分歧 → 保守派占上风 → 愤然辞职
- 进步派理念：社会制度应取决于生产力，魔导具与科学工程学的进步要求打破贵族对知识和教育的垄断，让有实力的人不再被血脉束缚
- 七位主教之间遵守传统以同门师兄师弟（师姐师妹）相称，彼此无血海深仇
- 其余主教尊重了阿尔维斯的选择，未曾追捕他

**莉莉安入学的交易**：
- 阿尔维斯写信给现任主教们请求让莉莉安入学
- 交换条件：阿尔维斯返回教廷参加秘密魔法研究项目
- 他的动机：1) 莉莉安有无尽的可能性，应获得更好的教育 2) 让贵族子弟看清楚血脉不是一切 3) 他的实力至上主义
- 返回教廷后对莉莉安在学院的遭遇全然不知，盲目自信徒弟的实力

**住处被毁的真相**：
- 其余主教做的痕迹消除工作——不想让阿尔维斯在外留下痕迹，并非针对莉莉安

**关系**：
- ← 莉莉安：弟子，在七年中教会了她魔法和生活，是改变她命运的人
- → 教廷：前主教之首 → 主动辞职 → 隐居 → 以交易方式返回（参与秘密研究）
- → 其他六位主教：曾经的同事，以同门师兄师弟相称，虽理念不合但无仇恨

**叙事功能**：
- 莉莉安的魔法导师和生活拯救者
- 教廷内部的知情人
- 莉莉安寻找师傅的悬念线（她不知道师傅在教廷）
- 可能与艾瑞安·赛拉斯被弑的真相有关
